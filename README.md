# PicSplit: 图片分割器

一个轻量级的在线图片分割工具，支持将图片均匀或自由分割成多块，并以 ZIP 格式下载。

## 功能

- **图片分割** — 自由设置行数和列数（1~50），将图片均匀分割成网格
- **拖拽分割线** — 直接拖动网格线自由调整分割位置
- **智能分割** — 基于背景色自动对齐分割线，适合分割图集/精灵图
- **边缘裁切** — 自动检测并裁切掉图片边缘的背景区域
- **格式选择** — 支持 PNG / JPEG / WebP 输出格式
- **批量下载** — 一键导出为 ZIP 压缩包，自定义文件名前缀
- **粘贴上传** — 支持 Ctrl+V 粘贴剪贴板中的图片
- **裁剪复原** — 裁切操作可撤销，支持多级回退

## 使用

```bash
# 启动服务
npm start

# 或直接
node server.js
```

打开浏览器访问 `http://localhost:3000`。

## 技术栈

- 纯前端（无框架依赖）
- Canvas API 处理图片
- JSZip 打包下载
- 单页应用，响应式布局
